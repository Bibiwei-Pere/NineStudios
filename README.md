## How to use

1. `Fork repository`

2. `Clone to vs code`

3. Open Command prompt and run `npm install`

4. Lastly run `npm start` and enjoy

2. UI Design: [Figma](https://www.figma.com/)

3. Code Editor: [VS Code](https://code.visualstudio.com/) and [Stackblitz](https://stackblitz.com/)
